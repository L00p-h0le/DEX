# DEX Contracts

Smart contracts for the Decentralized Exchange (DEX).

## Overview
Built with Solidity and Foundry. Includes core AMM mechanics (Factory, Pair), mock tokens for testing, and periphery Router contracts.

## Getting Started

### Prerequisites
- [Foundry](https://getfoundry.sh/)

### Build
```bash
forge install
forge build
```

### Test
```bash
forge test
```
